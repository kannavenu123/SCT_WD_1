# 🌐 Responsive Landing Page

A modern, responsive landing page built using **HTML, CSS, and JavaScript**. This project features a fixed navigation bar with a scroll effect, smooth hover animations, a responsive mobile menu, and a clean hero section.

---

## 📌 Project Overview

This landing page was created as part of the **SkillCraft Technology Web Development Internship - Task 1**.

The objective was to design a responsive landing page with an interactive navigation menu that changes its appearance while scrolling and works seamlessly across different screen sizes.

---

## ✨ Features

- Responsive design for desktop and mobile devices
- Fixed navigation bar
- Navbar background changes on scroll
- Glassmorphism navigation effect
- Animated hover underline on menu items
- Hero section with gradient text
- Responsive mobile navigation menu
- Smooth button hover effects
- Modern UI with gradients and animations

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)
